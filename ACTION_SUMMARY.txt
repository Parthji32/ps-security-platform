STEP 1: Install dependencies
  cd E:\demo
  npm install

STEP 2: Get Resend API key
  • Go to resend.io
  • Sign up (free, no credit card)
  • Copy your API key

STEP 3: Create .env file
  Create E:\demo\.env with:
  RESEND_API_KEY=re_xxxxxxxxxxxxx
  PORT=3001

STEP 4: Run backend locally
  node email-signup-backend.js

STEP 5: Test signup
  • Open index-with-signup.html in browser
  • Enter email and click "Join Early Access"
  • Check your email for welcome message ✅

STEP 6: Deploy to Render
  • Go to render.com
  • Create new Web Service
  • Connect GitHub repo
  • Add RESEND_API_KEY env var
  • Deploy

STEP 7: Update frontend
  In index-with-signup.html, change:
  const backendUrl = 'https://YOUR-RENDER-URL.onrender.com';

STEP 8: Push to GitHub
  git add .
  git commit -m "Add email signup system"
  git push origin main

STEP 9: Test live on GitHub Pages
  Visit: https://parthji32.github.io/ps-security-platform/
  Try signup form ✅
