Getting Started

Quick Start:

  1. Sign up for Kavach v1 (free tier)
  2. Create your first canary token
  3. Place it where attackers might find it
  4. Get alerted the moment it's triggered

Kavach v1

Multi-format canary tokens with real-time attacker fingerprinting.

Token types available:

  • URL Tokens
  • Document Tokens (Word, Excel)
  • API Key Tokens
  • DNS Tokens
  • Email Tokens

Kavach v2

ML-powered reverse proxy with 5-dimensional traffic classification.

Deploy via Docker Compose. Automatically routes traffic:

  • <31 risk score: Forward to real app
  • 31-70 risk score: Route to honeypot
  • 71-100 risk score: Block & log

Integration

  • Webhooks
  • Datadog
